# dictionary_hands_on



## 機能概要

自動翻訳機能付きの単語帳アプリです。

英→和/和→英の自動翻訳に対応しています。



![ss3](https://github.com/YujiOnishi/dictionary_hands_on/blob/master/movie1.gif)



## 開発環境に必要なもの一覧

AndroidStudio…プログラムを書くために必要

flutter…今回の主役。モバイルアプリ作成のために必要。

firebase…ミドルウェアに採択。DBと自動翻訳のAPIのために必要。
